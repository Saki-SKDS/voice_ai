---

## 🤔 COMMENT ÇA FONCTIONNE ? (2 minutes)

### Architecture technique :
- **Frontend** : Interface web HTML5/JavaScript
- **Backend** : Serveur Flask Python
- **IA Vocale** : 
  - Deepgram pour reconnaître la voix (STT)
  - ElevenLabs pour générer la voix (TTS)
  - OpenAI pour les réponses intelligentes

### Pipeline complet :
1. **Vous parlez** → Micro enregistre votre voix
2. **Deepgram** transforme votre voix en texte
3. **OpenAI** génère une réponse intelligente
4. **ElevenLabs** transforme la réponse en voix
5. **Vous entendez** la réponse de l'agent
