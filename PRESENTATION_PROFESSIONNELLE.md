## ⚙️ DÉMONSTRATION TECHNOLOGIQUE (3 minutes)

### Architecture robuste et scalable
```
Frontend (React/HTML5) → Backend Flask Python → Services IA
                                   ↓
                            Deepgram (STT) + ElevenLabs (TTS) + OpenAI
```

### Pipeline en temps réel
1. **Capture vocale** : Micro haute qualité avec réduction de bruit
2. **Reconnaissance** : Deepgram - 95% de précision en français
3. **Intelligence** : OpenAI GPT-3.5 pour réponses médicales validées
4. **Synthèse** : ElevenLabs - voix naturelle et empathique
5. **Feedback** : Temps de réponse < 2 secondes
